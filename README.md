# Geo
Program for creating area objects in OSM
Для запуска программы необходимо:
1. Клонировать репозиторий;
2. Создать в скачанном репозитории вирутальное окружение. Сделать это можно, перейдя в папку Geo и выполнив команду в CLI "python -m venv venv";
3. В той же директории ввести команду "pip install -r requirements.txt" для установки всех зависимостей;
4. Перейдите в директорию GeoMaps. Введите команду "python manage.py runserver" (Вместо команды "python" нужно использовать "python3", если ваша ОС не Windows);  
5. Перейдите в браузере на сайт по ссылке: "http://127.0.0.1:8000/cadastral/".


Инструкция:
1. Введите в поле кадастровый номер;
2. Нажмите кнопку "Найти";
3. После того, как на форме отобразиться время выполнения программы, нажмите на кнопку "Посмотреть", чтобы увидеть карту OSM с добавленным участком;
4. Закончите выполнение программы или переходитн к пункту 1.
